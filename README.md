# 💰 Budget Tracker

A clean, dark-themed budget tracking web app built with **HTML**, **Tailwind CSS**, and **vanilla JavaScript**.

---

## ✨ Features

- Add income and expense transactions
- Categorize transactions (Food, Salary, Transport, etc.)
- View net balance, total income, and total expenses at a glance
- Category breakdown bar chart for expenses
- Filter transactions by All / Income / Expenses
- Delete any transaction with a hover-to-reveal button
- Data persists in the browser using `localStorage`

---

## 🗂️ Project Structure

```
Budget_tracker/
├── index.html    # App structure and markup
├── style.css     # Custom styles and CSS variables
└── app.js        # All JavaScript logic
```

---

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/Amar-Fiaz/Budget_tracker.git
   ```

2. **Open the project folder**
   ```bash
   cd Budget_tracker
   ```

3. **Run the app**  
   Simply open `index.html` in your browser — no installation or build step needed!
